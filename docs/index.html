    General

    variables

    column-number

    $column-number: 12;

    Description

    The number of columns per row. This affects the width of columns created with the column mixin. Can be any whole number.

    gutter-width

    $gutter-width: 30px;

    Description

    The default width of gutters between columns. Can be any whole number with pixel units, but an even number is recommended, since half of the specified amount is applied to each side of columns as padding.

    mixins

    set-breakpoint

    @mixin set-breakpoint($prefix, $prefix-break: 0, $prefix-width: none) { ... }

    Description

    Creates a "breakpoint" for resizing columns with media queries. Creating a breakpoint will cause all rows will have a maximum width specified by that breakpoint. Not supplying the $prefix-break parameter (or setting it as 0) will set the minimum breakpoint (usually the "mobile" breakpoint).

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $prefix

    Prefix name that the breakpoint can be accessed by

    		String—none
    $prefix-break

    The minimum width of the screen, in pixels, for this breakpoint

    		Number0
    $prefix-width

    The max-width of the rows for this breakpoint

    		Numbernone

    Example

    // This breakpoint is the minimum breakpoint
@include set-breakpoint('xs')

// This breakpoint is only active when the screen is at least 768px wide,
// and the rows will have a max-width of 700px.
@include set-breakpoint('sm', 768px, 700px)

    Used by

    • [mixin] media

    media

    @mixin media($prefix) { ... }

    Description

    Mixin to alias media queries, based on the breakpoints created with the set-breakpoint mixin. Requires one or more breakpoints.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $prefix

    The name of the breakpoint

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    // Set a breakpoint to use
@include set-breakpoint('sm', 768px, 700px)

@include media('sm')
    // Rules here are applied when the device screen is at least 768px wide

    Throws

    row

    @mixin row($params...) { ... }

    Description

    Mixin to designate a selector as a row. Rows have a max-width that they inherit from the breakpoints created with the set-breakpoint mixin. Rows also have the wrap mixin (aka clearfix), so rows with columns will self-clear.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $params...

    Accepts multiple space-separated or comma-separated parameters. Currently, the only acceptable parameter is full, which causes the row not be limited by a max-width.

    		String—none

    Example

    // This row will change its max-width based on the screen size, based
// on the max-widths specified for the breakpoints
.example-class1
    @include row

// This row will not be limited by a max-width
.example-class2
    @include row(full)

    column

    @mixin column($width, $params...) { ... }

    Description

    Mixin to designate a selector as a column. Columns are not required to be placed within rows, but columns are floated, so their parent element needs to self-clear one way or another. Columns float to the left by default, and padding on the left and the right that is half of the $gutter-width.

    Accepted Values for $width:

    • (Number): Sets the width as being equal to ($width / $column-number).
    • full or whole: Sets the width as 100%.
    • half or one-half: Sets the width as 50%.
    • third or one-third: Sets the width as 33.33333%.
    • two-thirds: Sets the width as 66.66667%.
    • quarter or one-quarter: Sets the width as 25%.
    • two-quarters: Sets the width as 50%.
    • three-quarters: Sets the width as 75%.
    • '(Number) out of (Number)': Sets the widths as being equal to ($width / $column-number), by using the first (number) as $width and using the second (number) as $column-number. Any string or symbol (one or multiple) can be used as a separator between the two (number)s, as long as all are wrapped in single or double quotes to prevent any math symbols from changing the value before the function runs.

    Accepted $params:

    • center: Centers the column in its own row, removing the left float and clearing other columns.
    • right: Floats the column to the left, rather than the right.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    The width of the column. See description for accepted values.

    		Number or String—none
    $params...

    Additional parameters. See description for accepted values.

    		String—none
